# FEAT-002 - Conferência de estoque

Área: Estoque  
Tipo: Feature  
Prioridade: Média  
Status: Planejado  

## Descrição

Criar uma função para conferir estoque real versus estoque registrado no app.

## Funções possíveis

- iniciar conferência de estoque;
- listar itens;
- inserir quantidade real;
- registrar diferença como ajuste;
- criar histórico de ajuste;
- alerta semanal para revisão.

## Nome sugerido

Conferência de estoque.
