# Diet App - Visão Geral

## Objetivo

Aplicativo de dieta, treino, estoque, compras e progresso corporal focado em bodybuilding.

O app deve ajudar o usuário a:
- seguir uma dieta estruturada;
- registrar refeições consumidas;
- controlar estoque de ingredientes;
- gerar lista de compras;
- registrar treinos;
- acompanhar progresso corporal;
- analisar consistência, evolução e possíveis fatores de estagnação;
- futuramente usar IA para análises e alterações assistidas.

## Estratégia de desenvolvimento

Usar roadmap modular com ciclos pequenos.

Não usar waterfall gigante com um único prompt para construir tudo.
Não usar desenvolvimento caótico sem documentação.

Codex deve ser usado para tarefas delimitadas.
ChatGPT deve ser usado para arquitetura, revisão, planejamento, priorização e prompts.
