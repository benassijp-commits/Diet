# BUG-008 - Refeição avulsa com múltiplos itens

Área: Refeições / Consumo avulso  
Tipo: Feature/melhoria funcional  
Prioridade: Média  
Status: Planejado  

## Descrição

A refeição avulsa não deve registrar apenas um ingrediente por vez.

O usuário precisa montar uma refeição com vários itens no mesmo modal.

## Resultado esperado

Permitir adicionar múltiplos itens antes de salvar a refeição avulsa.
