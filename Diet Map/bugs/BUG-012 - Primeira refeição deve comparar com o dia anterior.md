# BUG-012 - Primeira refeição deve comparar com o dia anterior

Área: Histórico / Análise de refeição  
Tipo: Ajuste de regra de negócio  
Prioridade: Média  
Status: Planejado  

## Descrição

Hoje a primeira refeição compara o horário com a média histórica das primeiras refeições.

O comportamento desejado é comparar com o horário da primeira refeição do dia anterior.
