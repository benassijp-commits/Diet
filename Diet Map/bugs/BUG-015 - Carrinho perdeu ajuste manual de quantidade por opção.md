## Descrição

Na versão anterior, na aba de dieta/refeições, o carrinho permitia ajustar manualmente a quantidade de cada opção de refeição. O usuário conseguia adicionar e remover quantidades na parte inferior do carrinho.

Na versão atual, o carrinho apenas mostra o que está no carrinho, sem permitir ajustar/remover/adicionar quantidades diretamente por opção.

## Resultado atual

O carrinho exibe o resumo, mas não oferece controles manuais suficientes para alterar quantidades das opções.

## Resultado esperado

Restaurar controles para ajustar a quantidade de cada opção/refeição no carrinho.

Exemplo:
- Refeição 1 / Opção A: diminuir/aumentar quantidade
- Refeição 1 / Opção B: diminuir/aumentar quantidade
- Refeição 2 / Opção A: diminuir/aumentar quantidade

## Observação

Isso é regressão da migração React ou da restauração incompleta do fluxo antigo.
