# BUG-003 - Modal de edição fica cortado no celular

Área: UI Mobile / Refeições  
Tipo: Bug visual e funcional  
Prioridade: Alta  
Status: Confirmado  

## Descrição

No celular, o modal de edição de refeição às vezes fica cortado. Botões como cancelar, salvar e o topo perto do X podem ficar parcialmente fora da tela.

## Resultado esperado

O modal deve ser totalmente utilizável no celular, com rolagem adequada e botões sempre acessíveis.
