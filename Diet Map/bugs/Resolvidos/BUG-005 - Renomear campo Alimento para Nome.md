# BUG-005 - Renomear campo Alimento para Nome

Área: Refeições / Editor / UX  
Tipo: Melhoria de clareza  
Prioridade: Média  
Status: Planejado  

## Descrição

No editor de refeição, o campo “Alimento” confunde porque parece competir com o campo “Ingrediente”.

Na prática, esse campo representa o nome exibido para o usuário na refeição, podendo incluir preparo ou descrição.

## Mudança sugerida

Trocar o rótulo “Alimento” por “Nome”.
