# BUG-002 - Regressões da migração React

Área: Refeições / Migração React  
Tipo: Regressão  
Prioridade: Crítica  
Status: Confirmado parcialmente  

## Descrição

Após a migração para React, funcionalidades que existiam na versão main desapareceram ou ficaram incompletas na dev.

## Funcionalidades afetadas

- Colapsar/expandir cards de refeições desapareceu ou ficou instável.
- Editor de refeição parece travado na Opção A.
- Não há forma clara de editar Opção B/C.
- Função de adicionar nova opção de refeição desapareceu.
- Refeição avulsa desapareceu.
- Campo/configuração de intervalo mínimo entre refeições desapareceu.
- Modal de edição apresenta problemas visuais no mobile.

## Fora de escopo

- Não restaurar botão “reiniciar dia”.
- Não implementar features novas.
- Não mexer em tradução.
- Não mexer em offline avançado.
