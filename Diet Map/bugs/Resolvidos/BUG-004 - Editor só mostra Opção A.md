# BUG-004 - Editor só mostra Opção A

Área: Refeições / Editor  
Tipo: Bug funcional / regressão  
Prioridade: Alta  
Status: Confirmado  

## Descrição

Ao selecionar Opção B ou Opção C de uma refeição e tocar em editar, o editor continua mostrando apenas Opção A.

Antes da migração React, havia forma de selecionar qual opção da refeição editar.

## Resultado esperado

O modal de edição deve permitir escolher e editar qualquer opção da refeição: A, B, C ou opções adicionadas pelo usuário.
